História dos computadores

(Voltar)

Interessante saber a historia do computador, pois, a gente descobre como eram os primeiros computadores, modelos, tamanhos e o que faziam.

E ainda ver pessoas influentes nesse assunto, que fizeram diversas mudancas e criações para o computador ser o que é hoje

O que são Clients?

(Voltar)

Representa uma entidade que consome os serviços de uma outra entidade servidora.

Por exemplo, uma pessoa pesquisando sobre qualquer coisa na sua maquina, é o cliente do servidor

O que são Servers?

(Voltar)

O servidor é o computador ou software central que forma uma arquitetura conhecida como “cliente e servidor” com outros dispositivos, como computadores, tablets e smartphones. O servidor, portanto, centraliza todas as informações e demais dados dentro de uma rede.

Quando pesquisamos algo no navegador, o servidor fornece as informações em html e css.

Como é uma estrutura básica de um HTML?

(Voltar)

Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags html, head e body.

Onde html serve para comecar e encerrar o documento, o head sao informações do documento e body é o conteudo.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados.

A lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.
